# DMoney_API_Integration

## Project Description:
Dmoney is a Mobile Financial Service (MFS) platform (similar to bKash/Nagad) supporting four user roles: Admin, Agent, Customer, and Merchant. It consists of two sub-projects:

- `dmoney-transaction-api/` — Express.js REST API (backend)

- `dmoney-web-react/` — Next.js 15 frontend (TypeScript)

## How to run this project:
- Clone the Project
- open any Command Shell
- Give the following command `npm i` and `node .\report.js`
